# data.gov.lv (data-gov-lv)

data.gov.lv is a national government open-data portal for Latvia running **CKAN**, exposing a standardized catalog API over ~1,531 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/data-gov-lv/refs/heads/main/apis.yml)

**Run:** [Capabilities Using Naftiko](https://github.com/naftiko/fleet?utm_source=api-evangelist&utm_medium=readme&utm_campaign=data-gov-lv-api-evangelist&utm_content=repo)

## Type
- **kind:** government  ·  **software:** CKAN  ·  **version:** 2.11.3

## API
- **data.gov.lv CKAN Action API** — base `https://data.gov.lv/api/3/action`. [Docs](https://docs.ckan.org/en/latest/api/) · [Portal](https://data.gov.lv)
- Live endpoint verified: `https://data.gov.lv/api/3/action/status_show` · ~1,531 datasets
